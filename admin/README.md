# ChiqindiNav Admin

## Ishga tushirish

```
npm install
npm run dev
```

Admin panel API manzili `src/App.jsx` ichida `API_BASE` bilan boshqariladi.

Admin login:
- Backendda `CHIQINDI_ADMIN_USERNAME`, `CHIQINDI_ADMIN_PASSWORD` va `CHIQINDI_ADMIN_TOKEN` ni sozlang
- Admin panel login oynasida username/parol kiriting

Tuman admin yaratish:
- Super admin dashboardida login/parol va tumanni tanlab yaratish mumkin
